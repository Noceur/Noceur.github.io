Houdini - Shrinkwrap cliff 15.04.2023

One of my recent experiments was that I wanted to generate a base shape for a rock/cliff using some primitives or other shapes. The idea was to use a vellum solver to make a shrink wrap around the meshes to get the basic shape, and then apply some simple noises to make the shapes slightly more interesting. The current result is rather crude but with a better noise solution I think it could be pretty decent.

Finally there is a erosion system I put together using a fluid flip book solver as a test. The idea was to erode the top of the rock like if the weather has been breaking it down over the years.
